Notification Management API

A comprehensive REST API for managing notifications, templates, and delivery tracking

API Documentation

POST /api/signup

Create a new user account and receive an API key.

Request Body:
{
  "email": "user@example.com",
  "password": "securepassword"
}
Response:
{
  "message": "User created successfully",
  "email": "user@example.com",
  "api_key": "uuid-generated-key"
}

POST /api/services

Create a new service linked to your API key.

Headers:
X-API-Key: your-api-key
Request Body:
{
  "name": "My Service",
  "description": "Service description"
}

GET /api/services

Get all services for the authenticated user.

Headers:
X-API-Key: your-api-key

POST /api/templates

Create a new message template.

Headers:
X-API-Key: your-api-key
Request Body:
{
  "name": "Welcome Email",
  "subject": "Welcome to Our Service!",
  "body": "Hello! Welcome to our amazing service.",
  "service_id": "service-uuid"
}

GET /api/templates

Get all templates for the authenticated user.

Headers:
X-API-Key: your-api-key

POST /api/notifications/email

Send an email notification using a template.

Headers:
X-API-Key: your-api-key
Request Body:
{
  "template_id": "template-uuid",
  "recipient_email": "recipient@example.com"
}

POST /api/notifications/sms

Send an SMS notification using a template.

Headers:
X-API-Key: your-api-key
Request Body:
{
  "template_id": "template-uuid",
  "recipient_phone": "+1234567890"
}

GET /api/messages/{message_id}/status

Get the status of a previously sent message.

Headers:
X-API-Key: your-api-key
Response:
{
  "message_id": "message-uuid",
  "status": "delivered",
  "type": "email",
  "recipient": "recipient@example.com",
  "sent_at": "2023-01-01T00:00:00",
  "delivered_at": "2023-01-01T00:00:05"
}

GET /api/messages

Get all messages for the authenticated user.

Headers:
X-API-Key: your-api-key
